### Prerequisites
- Python 3.8+

- Google Earth Engine account (for full features)

### Get Started
```bash
git clone https://github.com/yourusername/flutter-earth.git
cd flutter-earth
```

#### Install Python dependencies
```bash

```

#### Install frontend dependencies
```bash
cd frontend

```

#### Run the app
```bash
cd ..
run_desktop.bat
```
