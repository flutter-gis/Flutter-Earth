## ‚ú® What is Flutter Earth?

Flutter Earth is your all-in-one, supercharged, desktop playground for:
- üõ∞Ô∏è **Exploring satellites** (with real-time data crawling!)
- üì• **Downloading Earth Engine data** (with a beautiful progress bar!)
- üå± **Running index analysis** (NDVI, NDWI, and more!)
- üé® **Switching between 20+ wild themes** (from Minecraft to Pride!)
- üåê **Viewing and exporting raster/vector data**
- ‚ö° **Having fun while doing serious science!**

## üöÄ Quickstart (It's Easy!)

<div class="code-block">
```bash
# 1. Clone the repo
$ git clone https://github.com/yourusername/flutter-earth.git
$ cd flutter-earth

# 2. Install dependencies
$ cd frontend && npm install
$ cd ../backend && pip install -r requirements.txt

# 3. Launch the app
$ cd .. && npm start
```

## üõ†Ô∏è Setup & Configuration

- **Node.js** v16+ and **Python** 3.8+
- (Optional) [Google Earth Engine](https://earthengine.google.com) account for full power
- Place your Earth Engine credentials in `backend/`

### Custom Themes? Heck Yes!
Add your own to `frontend/themes.json`:
```json
{
  "name": "my_theme",
  "display_name": "My Custom Theme",
  "category": "custom",
  "background": "#1a1a1a",
  "primary": "#4CAF50",
  "text": "#ffffff",
  "splashEffect": "confetti"
}
```
