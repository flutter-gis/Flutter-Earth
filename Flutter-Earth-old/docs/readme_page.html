About Flutter Earth

Flutter Earth is a powerful desktop application that provides easy access to Google Earth Engine's vast collection of satellite imagery and geospatial data. Built with Electron and Python, it offers a modern, user-friendly interface for downloading and processing satellite data.

Getting Started

Prerequisites

  • Google Earth Engine account and authentication
  • Python 3.8 or higher
  • Node.js 16 or higher
  • Internet connection for data access

Installation

  1. Clone or download the Flutter Earth repository
  2. Install Python dependencies:
    pip install -r requirements.txt
  3. Install Node.js dependencies:
    cd frontend && npm install
  4. Set up Google Earth Engine authentication

Running the Application

Use one of the following methods to start Flutter Earth:

  • Windows: Double-click run_desktop.bat
  • PowerShell: Run .\run_desktop.ps1
  • Manual: Navigate to frontend/ and run npm start

Usage Guide

Basic Workflow

  1. Authentication: Set up your Google Earth Engine credentials
  2. Map Selection: Choose your area of interest on the interactive map
  3. Satellite Selection: Select the satellite dataset you want to download
  4. Parameters: Configure date range, cloud cover, and other parameters
  5. Download: Start the download and monitor progress

Advanced Features

  • Index Analysis: Calculate NDVI, NDWI, and other vegetation indices
  • Vector Downloads: Download administrative boundaries and other vector data
  • Batch Processing: Process multiple areas or time periods
  • Data Export: Export data in various formats (GeoTIFF, CSV, etc.)

Themes and Customization

Flutter Earth comes with a variety of themes to personalize your experience. From professional dark and light themes to fun character-inspired themes, you can choose the style that suits you best.

Tip: You can switch themes in the Settings panel and customize theme options like animations and special effects.

Troubleshooting

Common Issues

  • Authentication Errors: Ensure your Google Earth Engine credentials are properly set up
  • Download Failures: Check your internet connection and available disk space
  • Performance Issues: Close other applications to free up system resources
  • Theme Issues: Try resetting to the default theme if custom themes cause problems

Getting Help

If you encounter issues not covered here, please check the logs in the logs/ directory for detailed error information. You can also clear the cache and logs through the Settings panel.

Technical Details

Architecture

Flutter Earth uses a hybrid architecture combining Electron for the user interface and Python for Earth Engine operations. This provides the best of both worlds: a modern, responsive UI and powerful geospatial processing capabilities.

Technologies Used

  • Frontend: HTML5, CSS3, JavaScript, Electron
  • Backend: Python, Google Earth Engine API
  • Data Processing: NumPy, Pandas, GDAL
  • UI Framework: Custom CSS with theme system
